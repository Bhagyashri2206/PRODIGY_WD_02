# PRODIGY_WD_02

 Neon Stopwatch Web App

A modern **stopwatch web application** built using **HTML, CSS, and JavaScript** with a sleek **neon glassmorphism UI** and smooth animations.

---

 Features
- Start, Pause, Reset stopwatch
- Lap time recording with time difference
- Fastest & slowest lap highlighting
- Neon glow animation while running
- Light / Dark theme toggle
- Keyboard shortcuts support
- Sound effects for actions
- Persistent lap data using LocalStorage

---

 Tech Stack
- HTML5  
- CSS3 (Animations, Glassmorphism)  
- JavaScript (ES6)
